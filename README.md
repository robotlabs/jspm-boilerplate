# jspm-boilerplate
my boilerplate, using jspm

install jspm if you haven't, globally. ( npm install -g jspm@beta if you fancy go beta)

### install
`make setup`

### build
`make build m=dev`

`make build m=prod`

###watch
`make watch m=dev`

`make watch m=prod`

### build for release
`make build m=prod q=release`

### watch for release
`make watch m=prod q=release`
